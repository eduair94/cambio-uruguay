## How backlinks actually help (read once, set expectations)

Most links below are **nofollow** (Reddit, HN, Quora, new dev.to/Hashnode accounts, many directories). Nofollow ≠ useless:
- **Referral traffic** — real visitors, the point of the whole exercise.
- **Discovery + brand signals** — Google/AI see mentions, indexing crawlers follow.
- **Some flip dofollow** — dev.to/Hashnode go dofollow as your account gains reputation.

**Dofollow (real link equity)** comes from: GitHub awesome-list PRs, `public-apis` PR, some directories (AlternativeTo, SaaSHub), guest posts, and editorial/journalist links. Those are the ones worth extra effort.

Don't spam. A currency-comparison tool posted as genuine help in the right thread = link + traffic. Same thing carpet-bombed = removed + account banned. Ratio: mostly contribute, occasionally link.

## Priority order (do in this order)

1. **Tier 1 registries** — you have an MCP server + public API + npm package. Free, high-authority, near-zero rejection. Biggest ROI. Start here.
2. **Dev.to + Hashnode cross-posts** — content already written (Medium articles). 20 min each.
3. **App directories** (AlternativeTo, SaaSHub, ProductHunt, Uneed, Fazier).
4. **Show HN + Reddit** — one shot each, timing matters (see tier2).
5. **Forums + Quora** — ongoing, answer real questions.
6. **Journalist/PR** — highest-value links, slowest. Background task.
